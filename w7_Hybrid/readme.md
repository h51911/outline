# Hybrid

## 应用分类（App）
* WebApp
    * html+css+js
    * H5: 前端
    * 特点：
        * 跨平台
        * 成本低
    * js是一门客户端语言
* NativeApp    原生应用
    * 平台
        * iOS
        * android
        * winPhone
        * 塞班
        * 黑莓
        * ....
    * 特点
        * 成本高
        * 不能跨平台

        * 流畅
        * 硬件调用
* HybridApp 混合应用

* 性能过剩
